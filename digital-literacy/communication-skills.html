Communication Skills 💬

Good communication helps us learn, work, and build strong friendships. Whether online or offline, how we speak shows our character.

Good Message vs Bad Message

❌

Bad Message

SEND ME THAT FILE NOW!!!

This sounds rude, angry, and disrespectful.

✅

Good Message

Hello! Please can you send me the file when you are free? Thank you 😊

This sounds polite, kind, and professional.

Writing a Proper Email 📧

4 Parts of an Email

  • Subject: Short reason for writing
  • Greeting: Dear Teacher
  • Body: Explain clearly
  • Closing: Thank you + your name

Example

Subject: Question About Homework

Dear Teacher,

Please I need help with Question 3 in today’s assignment.

Thank you.
John

The 3-Second Rule ⏳

Before you send any message, ask: Is it Kind? Is it True? Is it Necessary?
