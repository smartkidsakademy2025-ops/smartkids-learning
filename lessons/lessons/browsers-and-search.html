Module 2: The Web

Browsers & Search Engines

Your tools for exploring the vast world of the internet.

🌍

The Browser

The Browser is the software application you use to access and view websites.

Examples:

  • Google Chrome
  • Safari
  • Microsoft Edge
🔍

The Search Engine

A Search Engine is a special website that helps you find other websites.

Examples:

  • Google Search
  • Bing
  • DuckDuckGo

How they work together:

"Think of the **Browser** as a car. It is the vehicle that takes you where you want to go. The **Search Engine** is the map or GPS. You use the map to find the location, but you need the car to actually drive there!"

Staying Safe While Searching

When searching for things online, always remember:

  • ✅ Use Kid-Safe search engines like Kiddle.
  • ✅ Never click on strange pop-up ads.
  • ✅ Tell a parent if you see something that makes you uncomfortable.
